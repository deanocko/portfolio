Automatic Aircraft Visual Inspection (AAVI)

Project Overview

The Automatic Aircraft Visual Inspection (AAVI) project represents a significant advancement in automated defect detection for aircraft maintenance. Using the state-of-the-art YOLO-NAS (You Only Look Once - Neural Architecture Search) model, this system achieves unprecedented accuracy in identifying various types of aircraft defects.

Technical Implementation

Key technical aspects of the project include:

  • Implementation of YOLO-NAS CNN architecture
  • Custom dataset creation and annotation
  • Real-time defect detection capabilities
  • Edge device deployment optimization
  • Comprehensive performance evaluation framework

Results and Impact

The project achieved several significant outcomes:

  • Best-in-class detection accuracy compared to existing literature
  • Real-time processing capabilities suitable for industrial deployment
  • Successful validation on diverse aircraft defect types
  • Potential for significant cost reduction in aircraft maintenance

Future Development

Ongoing and future development areas include:

  • Integration with automated maintenance systems
  • Expansion of the defect detection categories
  • Enhancement of edge device performance
  • Development of a comprehensive deployment framework
