Automatic Aircraft Visual Inspection

Computer Vision & Deep Learning

Summary

  • Applied YOLO-NAS CNN model to automatically detect defects in aircraft imagery
  • Achieved the best results in the current literature for aircraft defect detection
  • Developed a system capable of real-time deployment on edge devices

Project Overview

The Automatic Aircraft Visual Inspection (AAVI) project represents a significant advancement in automated defect detection for aircraft maintenance. Using state-of-the-art YOLO-NAS (You Only Look Once - Neural Architecture Search) technology, this system achieves unprecedented accuracy in identifying various types of aircraft surface defects.

This innovative solution addresses the critical need for efficient and reliable aircraft inspection processes, potentially reducing inspection time while maintaining or improving accuracy.

Our approach involved several key steps including dataset preparation with diverse aircraft surface defect images, implementation of YOLO-NAS architecture for defect detection, model training with optimized hyperparameters, extensive testing and validation on real-world scenarios, and thorough performance analysis and optimization.

Results

The AAVI system demonstrated exceptional performance in defect detection, achieving the highest accuracy rates reported in the current literature. The system successfully demonstrated high accuracy in identifying various types of surface defects, real-time processing capabilities, robust performance under different lighting conditions, and significant reduction in inspection time compared to manual methods. These results validate the effectiveness of our approach and its potential for real-world applications in aircraft maintenance and safety inspection procedures.

Potential Improvements

Future enhancements could include expanding the system to handle a wider variety of aircraft types and defect categories, implementing automated severity assessment for detected defects, developing a more sophisticated tracking system for defect progression over time, and integrating the system with existing maintenance management software for seamless workflow integration.
