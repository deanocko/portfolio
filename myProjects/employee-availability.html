Employee Availability

A DIY solution to manage employee work schedules using Google Sheets
Project Overview
I created this project when one of my workplaces wasn't satisfied with the employee scheduling software available on the market as they wanted us to be able to work together but couldn't see at a glance when everyone was available. As such, I created my own using a shared interactive Google Sheet.
